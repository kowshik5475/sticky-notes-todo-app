# Sticky Notes To-Do App

A responsive Sticky Notes themed To-Do List application built using HTML, CSS, and JavaScript.

## Features

- Create, Edit, Delete Tasks
- Mark Tasks as Completed
- Filter Tasks (All, Active, Completed)
- Persistent Storage using localStorage
- Inline Task Editing
- Clear Completed Tasks
- Responsive Design
- Accessibility Improvements

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## Concepts Demonstrated

- DOM Manipulation
- Event Delegation
- State Management
- CRUD Operations
- Client-Side Data Persistence
